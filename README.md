# Electricista 380

Landing page profesional de alta conversión y optimización SEO, diseñada bajo los lineamientos de AAIERIC para servicios eléctricos en Argentina.

## Arquitectura
- **Backend:** FastAPI
- **Contenido:** YAML dinámico (/data)
- **Frontend:** SSR con Jinja2 y JS modularizado.
- **Automatización:** Integración con Chatwoot y soporte preparado para bot inteligente.
